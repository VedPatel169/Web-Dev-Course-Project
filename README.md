# Web-Dev-Course-Project

## How to Run

1. **Navigate to the project directory**
    ```bash
    cd Web-Dev-Course-Project/study-app
    ```

2. **Start the server**
    ```bash
    node server.js
    ```

3. **Open the app in your browser**  
    - Visit: http://localhost:3000/api/sessions

4. **Open the frontend**
    - Locate and open `index.html` in your browser

5. **Use the app**
    - Enter your data parameters  
    - Add a session

6. **View updates**
    - Refresh the browser to see the newly added data

7. **Stop the server**
   - Press `Ctrl + C` in the terminal
